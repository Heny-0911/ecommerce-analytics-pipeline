# E-commerce Analytics Project

## About
This project demonstrates a **complete e-commerce data workflow** using Python. It includes:

- ETL pipeline: load, clean, and merge customer, product, and order data
- KPI calculations: total revenue, monthly trends, top products and customers, repeat customer rate, and customer lifetime value (CLV)
- Customer segmentation using RFM (Recency, Frequency, Monetary)
- Cohort analysis and retention rate

> ⚠️ Note: All data is **simulated** for demonstration purposes.
>
> ## Project Files

- `pipeline.py` — ETL pipeline code
- `kpi.py` — KPI calculation code
- `rfm_cohort.py` — RFM and Cohort analysis code
- 'visualization.py' - visualization code

## Tools Used

- Python (Pandas, NumPy, Matplotlib, Seaborn)

- SQLite

- Google Colab (for development)
